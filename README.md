# AutoFocus

## What this is

AutoFocus is a simple web app that helps you stay productive. It uses a timer to help you work in short focus sessions and tracks your progress with streaks and XP.

The goal is to make it easy to start working without any setup or distractions.

## Problem

Most productivity apps are too complicated. They have too many features, settings, or steps before you can start working.

This project keeps it simple so you can just open it and focus.

## Features

### Timer

A basic timer with start, pause, and reset buttons. You use it for focus sessions.

### Streaks

Tracks how many sessions you complete in a row.

### XP and Levels

You earn XP every time you finish a session. After enough XP, you level up.

### Stats

Shows your total XP and total minutes focused.

### Tabs

The app is split into simple sections:

* Home
* Timer
* Stats
* Settings

## Design

The design is soft and minimal with a light pink style. It is made to feel calm and not distracting.

## How to use

1. Open the app
2. Go to the Timer tab
3. Start a focus session
4. Work until the timer ends
5. Earn XP and build your streak
6. Check your progress in Stats
